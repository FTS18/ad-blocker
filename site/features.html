Feature Showcase

18 Star Features

Explore the detailed engineering capabilities that power our lightweight ad blocking extension.

1. Static Display Ad Blocking

Declarative net request rulesets block standard advertising banners, display units, and native placements before they can perform network queries.

2. Popunder & Redirect Protection

Blocks spam domains, popunder targets, and automatic page navigation redirects from malicious networks.

3. Path-Based Ad Script Blocks

Deep path filters block ad-serving scripts like /invoke.js even if hosted on CNAME-cloaked domains.

4. Google Analytics Redirection

Intercepts and redirects Google Analytics / Tag Manager request targets to secure local empty stubs (ga.js), preventing page crashes.

5. Telemetry & Tracking Blocks

Identifies and filters outbound tracking queries sent to Hotjar, Mixpanel, CrazyEgg, and enterprise data platforms.

6. Social Pixel Mitigation

Blocks third-party social tracking pixels (Facebook, Snapchat, LinkedIn, TikTok) loaded inside script tags and sub-frames.

7. Anti-Adblock Defeater

Blocks detection libraries like BlockAdBlock.js to stop anti-adblock modals from loading and prompting users.

8. Cryptominer Blocker

Filters CPU-draining browser cryptominers (Coinhive, WebMinerPool) to prevent unauthorized background coin mining.

9. Phishing & Scam Shield

A fourth ruleset layer blocking over 70 known malicious phishing domains, lottery scams, and tech-support fraud hosts.

10. HTTP-to-HTTPS Upgrades

DNR automatically upgrades unencrypted http requests to secure https for safer web browsing.

11. Real-Time Badge Counters

Automatically logs blocked requests per active tab and displays an action count directly on the extension icon badge.

12. Deterministic Whitelist Hashes

Maps site hostname strings to stable rule IDs using a deterministic hash function, avoiding ID collisions.

13. DOM Element Picker

Allows users to visually select elements with an overlay and hide them permanently from the layout on click.

14. Settings Option Dashboard

A clean user settings panel to manage trusted whitelisted sites, custom block rules, and write CSS cosmetic overrides.

15. WebAccessible Resource Stubs

Ensures redirection stubs are securely registered, allowing safe local scripts to bypass browser security boundaries.

16. WebRTC IP Leak Protection

Blocks WebRTC UDP queries from disclosing local IP addresses, preserving location privacy.

17. Smartlink Sanitizer

Intercepts link clicks on the capture phase, automatically decoding and cleaning affiliate redirect parameters.

18. JS Cookie Guard

Intercepts calls to document.cookie and blocks tracking cookie writes (_ga, _fbp) while preserving session credentials.
